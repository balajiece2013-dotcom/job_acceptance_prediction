# 🎯 HR Analytics: Candidate Placement Prediction System

## 📌 Project Overview

This project is a Machine Learning-based HR Analytics system designed to predict candidate placement outcomes using academic performance, technical skills, interview scores, and experience-related features.

The system helps recruitment teams analyze placement patterns, identify important hiring factors, and improve hiring decision-making using predictive analytics.

---

# 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Multiple Machine Learning Models
- Model Comparison & Evaluation
- Streamlit Interactive Dashboard
- Candidate Placement Prediction
- Feature Importance Visualization

# ⚙️ How to Run the Project

## 1️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

## 2️⃣ Run Model Training

```bash
python scripts/model_training.py
```

## 3️⃣ Run Streamlit App

```bash
streamlit run app/app.py
```
